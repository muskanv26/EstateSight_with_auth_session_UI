
EstateSight — Interactive House Price Prediction (Frontend)
--------------------------------------------------------

Files included:
- index.html
- styles.css
- app.js
- images/ (sample images and logo.svg)

How to run:
1. Open index.html in a modern browser.
2. To connect to a real ML model, replace the `predict` function in app.js with a fetch() call to your API or integrate TFJS/ONNX.

Logo:
- A simple SVG logo is included at images/logo.svg. You can replace it with a high-resolution version.
